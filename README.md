# ProjectFrameMvpOpt



基于android平台，采用mvp架构，实现项目基本框架。集成包括androidx、retrofit2、glide4.5、Luban1.8、mmkv1.3.5、room2.3.0等开源框架

实现功能如下：
网络请求封装
持久化MMKV，替换SharePreferences方案
数据库room操作封装
WebView封装，支持原生应用同vue、uni小程序交互；支持图片选取及压缩

其他功能陆续集成开发中...
